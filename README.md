# GeoNER-corpus

## Sources and contributors

Sources of the corpus:
* Simon Gabay (éd.), PARALLEL17: a parallel corpus (diplomatic vs normalised) of 17th c. French texts, Neuchâtel: Université de Neuchâtel, 2019, https://github.com/e-ditiones/PARALLEL17.
* https://fr.wikisource.org/wiki/M%C3%A9moires_et_Lettres_de_Marguerite_de_Valois/Les_M%C3%A9moires_de_Marguerite_de_Valois

Direct and indirect contributors: Simon Gabay, Philippe Gambette, Eleni Kogkitsidou, Jonathan Poinhos, Isabelle Qin, Caroline Trotot, Éliane Viennot

Derived from:
